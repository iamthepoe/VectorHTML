O que é o HTML?

O HTML (HyperText Markup Language) é uma linguagem de marcação. É através dos seus códigos que o conteúdo é interpretado, assim criando uma página na Internet. Esses códigos inseridos em tags são responsáveis por definir todos os elementos presentes na estrutura de uma página, como o seu tamanho, a fonte da letra, as suas cores, as quebras de linhas, dentre outros.

O surgimento do HTML

A linguagem HTML foi criada por Tim Berners-Lee e seus colegas pesquisadores entre 1990 e 1995, com o objetivo de ter uma maneira mais fácil de compartilhar documentos científicos. O seu principal objetivo era inventar algo parecido com um repositório hospedado na rede com todos os arquivos científicos que eles quisessem compartilhar. Para isso seria necessário uma linguagem que fosse até o servidor (repositório) e que apresenta-se o resultado da busca no navegador. Como resultado, essa linguagem teria como objetivo realizar a comunicação entre o servidor (server-side) e o navegador (client-side). Isto resultou nas páginas da web que vemos no navegador.

As versões de HTML

O HTML passou por várias atualizações desde o seu surgimento na década de 90. As suas versões são:


Versão 2.0 de 1995:

A primeira versão que surgiu após a criação do HTML puro foi a versão 2.0. Ela tinha como objetivo formalizar todas as características do código HTML que já eram utilizadas. Sua estreia aconteceu no primeiro evento mundial sobre a Web, chamado World Wide Web Conference.


Versão 3.2. de 1997:

Está versão do HTML proporcionou a criação de tabelas, applets e textos flutuantes ao redor de imagens, além da correção de problemas de compatibilidade da versão anterior.


Versão 4.01 de 1999:

A principal característica desta versão era fornecer compatibilidade com as versões anteriores, mas também era possível acrescentar suporte para opções de multimídias, folhas de estilos e melhorar as práticas da programação.
